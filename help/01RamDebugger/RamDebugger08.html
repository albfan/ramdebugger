Timing control

An additional feature of RamDebugger is the possibility of controlling the computing time of several parts of the program.

To proceed with the control of time, make the following steps:

  • Load the TCL file inside RamDebugger with Files->Open file . In this case, use the example located in <RamDebugger directory>/Examples/example3.tcl
  • Open window Debug->Timing control
  • Create a first block with name Block 1 , initial line 26 and end line 27 .
  • Create another block with name Block 2 , initial line 26 and end line 26 .
  • Create another block with name Block 3 , initial line 27 and end line 27 .
  • Press button Start and select Files->Debug on->Currentfile .
  • Check the report with button Report
