Revisions

RamDebugger uses program cvs in order to maintain revisions of the edited files. It creates automatically a CVS repository in a hidden directory and saves revisions of the files to that repository.

It is possible to visualize the differences between two revisions or between one revision and current file by using menu command: Files->Revisions->Open revisions list. (it uses program Tkdiff to visualize the differences.

It is possible to remove revisions history for a set of selected files (in Files->Revision-> View revised files). The history is removed but the file is not modified. It can save disk space.

It is possible to activate or deactivate the Auto save feature in the preferences window (Edit->Preferences->Auto save). If activated, instead of making backups of the files it saves automatically revisions of the edited file at fixed intervals.
