vcs-ramdebugger

vcs-ramdebugger

is the RamDebugger version control system GUI as independent program.

It can be used to make the most common operation in a VCS system like fossil or cvs (commit, update, add, remove, etc). Other less common operations like creating, cloning or opening the repository must be performed outside of vcs-ramdebugger, by using the command line.
